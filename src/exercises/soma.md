# Soma de Inteiros

Crie uma função chamada `somar($a, $b)` que receba dois números inteiros e retorne a soma deles.

## Exemplo:

```php
somar(2, 3); // deve retornar 5
```

## Requisitos:

- A função deve se chamar `somar`.
- Deve aceitar dois parâmetros.
- Deve retornar um número inteiro.
