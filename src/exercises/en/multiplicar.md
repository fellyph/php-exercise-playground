# Multiply Integers

Create a function named `multiply($a, $b)` that receives two integers and returns their product.

## Example:

```php
multiply(2, 3); // should return 6
```

## Requirements:

- The function must be named `multiply`.
- It must accept two parameters.
- It must return an integer.
