# Suma de Enteros

Crea una función llamada `sumar($a, $b)` que reciba dos números enteros y devuelva su suma.

## Ejemplo:

```php
sumar(2, 3); // debe devolver 5
```

## Requisitos:

- La función debe llamarse `sumar`.
- Debe aceptar dos parámetros.
- Debe devolver un número entero.
