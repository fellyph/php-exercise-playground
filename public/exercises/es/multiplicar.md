# Multiplicación de Enteros

Crea una función llamada `multiplicar($a, $b)` que reciba dos números enteros y devuelva su producto.

## Ejemplo:

```php
multiplicar(2, 3); // debe devolver 6
```

## Requisitos:

- La función debe llamarse `multiplicar`.
- Debe aceptar dos parámetros.
- Debe devolver un número entero.
