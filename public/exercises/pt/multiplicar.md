# Multiplicação de Inteiros

Crie uma função chamada `multiplicar($a, $b)` que receba dois números inteiros e retorne o produto deles.

## Exemplo:

```php
multiplicar(2, 3); // deve retornar 6
```

## Requisitos:

- A função deve se chamar `multiplicar`.
- Deve aceitar dois parâmetros.
- Deve retornar um número inteiro.
