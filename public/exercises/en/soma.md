# Sum of Integers

Create a function named `sum($a, $b)` that receives two integers and returns their sum.

## Example:

```php
sum(2, 3); // should return 5
```

## Requirements:

- The function must be named `sum`.
- It must accept two parameters.
- It must return an integer.
