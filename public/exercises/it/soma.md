# Somma di interi

Crea una funzione chiamata `somma($a, $b)` che riceva due numeri interi e restituisca la loro somma.

## Esempio:

```php
somma(2, 3); // dovrebbe restituire 5
```

## Requisiti:

- La funzione deve chiamarsi `somma`.
- Deve accettare due parametri.
- Deve restituire un numero intero.
