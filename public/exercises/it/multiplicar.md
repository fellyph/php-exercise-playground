# Moltiplicazione di interi

Crea una funzione chiamata `moltiplicare($a, $b)` che riceva due numeri interi e restituisca il loro prodotto.

## Esempio:

```php
moltiplicare(2, 3); // dovrebbe restituire 6
```

## Requisiti:

- La funzione deve chiamarsi `moltiplicare`.
- Deve accettare due parametri.
- Deve restituire un numero intero.
