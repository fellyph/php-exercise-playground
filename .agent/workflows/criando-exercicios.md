## Criando exercícios

1. Analise o exercício
   1. Como criar o desafio
   2. Crie exemplos simples
   3. Verifique qual é o nível do exercício: básico, intermediário e avançado.
2. Idiomas
   1. Criar versões em português, inglês e espanhol
3. Testar o exercício
   1. Sempre que criar um exercício valide uma resposta correta e incorreta para verificar a validação
4. Documente o novo exercício
   1. Liste no arquivo readme todos os exercícios disponíveis
